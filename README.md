## About Me

AI를 실제 도구와 운영 흐름에 연결하는 프로젝트를 만들고 있습니다.

관심사는 모델을 단순히 호출하는 것보다, Linux 모니터링, 로컬 LLM 분석, Docker 상태 점검, Windows 검수 자동화처럼 반복 작업을 줄이고 판단을 빠르게 만드는 시스템을 구현하는 것입니다.

```txt
Current focus
├─ Local AI tools with Ollama
├─ Linux monitoring and server operation
├─ Docker-based backend services
├─ Windows inspection and QC automation
└─ Practical portfolio projects with docs and reports
```
