## Featured Projects

### [Linux System Monitor](https://github.com/jiyoon99/linux-system-monitor)

FastAPI, Docker, nginx, Chart.js, Ollama를 조합한 Linux 서버 모니터링 대시보드입니다.

- CPU, RAM, Disk, Network 실시간 모니터링
- Docker daemon, container, image 상태 확인
- Server Bot 자동 점검과 WARN/FAIL 리포트
- Ollama 로컬 LLM 상태 확인 및 시스템 분석
- 웹 대시보드와 CLI 동시 지원

### [Linux Desktop Widget](https://github.com/jiyoon99/linux-desktop-widget)

Linux 데스크톱에서 Docker, 로컬 서비스, Ollama 상태를 확인하는 GTK 위젯입니다.

- Docker daemon 및 실행 중인 컨테이너 상태 확인
- 로컬 포트와 서비스 상태 점검
- Ollama API와 설치 모델 상태 확인
- 3초 간격 자동 갱신
- 로그인 자동 실행 스크립트 제공

### [Linux Chess Portfolio](https://github.com/jiyoon99/linux-chess-portfolio)

Go, WebSocket, PostgreSQL, Redis, Docker를 사용한 실시간 체스 서비스 포트폴리오입니다.

- REST API와 WebSocket 기반 실시간 대국
- 서버 측 체스 규칙 검증
- 회원가입, 로그인, 게임 기록, PGN 리뷰
- Docker Compose 기반 실행 환경
- nginx/Caddy, Prometheus, Grafana, Alertmanager 운영 문서

### Windows Automation Tools

[inspection-automation](https://github.com/jiyoon99/inspection-automation) /
[driver-check-helper](https://github.com/jiyoon99/driver-check-helper) /
[usb-qc-automation](https://github.com/jiyoon99/usb-qc-automation)

- CSV/HTML/PDF 리포트 생성
- 반복 검수 업무를 자동화 도구로 정리
